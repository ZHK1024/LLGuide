## Installation

To install it, simply add the following line to your Podfile:

```ruby
pod 'LLGuide', :git => 'https://github.com/ZHK1024/LLGuide.git'

pod 'LLGuide', :git => 'https://github.com/ZHK1024/LLGuide.git', :tag => s.version.to_s
```
